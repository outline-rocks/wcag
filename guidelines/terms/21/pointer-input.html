Zeigereingabe (pointer input)

Eingabegerät, das eine bestimmte Koordinate (oder einen Satz von Koordinaten) auf einem Bildschirm anvisieren kann, wie z. B. eine Maus, ein Stift oder ein Berührungskontakt (touch contact)

Siehe auch Pointer Events Zeigerdefinition (Pointer Events pointer definition) [[pointerevents]].
